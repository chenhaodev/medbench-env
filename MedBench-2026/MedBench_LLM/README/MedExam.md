## 介绍

MedExam数据集分为基础科目、专业科目和公共科目三部分。题目全部采用客观性选择题，设计有K型（最佳选择题）和S型题（病历摘要型最佳选择题）两类题，以下为两类题型的简述：

-   K型题（单句型最佳选择题）：每道试题由1个题干和5个供选择的备选答案组成。题干以叙述式单句出现，备选答案中只有1个是最佳选择，称为正确答案，其余4个均为干扰答案。
-   S型题（病历摘要型最佳选择题）：试题结构是由1个简要病历作为题干、5个供选择的备选答案组成，备选答案中只有1个是最佳选择。

该任务的评测采用精准度Accuracy作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：医学考试相关问题
answer：答案
</p></div>
```
