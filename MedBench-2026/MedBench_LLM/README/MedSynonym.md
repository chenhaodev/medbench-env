## 介绍

MedSynonym数据集面向医学专业人士，要求结合多场景（涵盖临床诊疗中的问诊、病历书写、手术沟通、检验报告解读，医学科研中的文献撰写、数据统计、实验记录，学术交流中的会议报告、病例讨论、教材编写，以及医患沟通中的病情说明、诊疗方案解释等），为核心医学专业术语匹配精准同义词，确保同义词符合权威医学标准，且适配具体场景的专业表达需求。任务涉及的术语涵盖基础医学、临床诊断、疾病病理、治疗干预、药学、检验医学等6大领域。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：核心医学专业术语及对应场景
answer：同义词
</p></div>
```
