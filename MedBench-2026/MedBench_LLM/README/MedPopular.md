## 介绍

MedPopular数据集聚焦于评估模型高质量医学科普内容生成的能力，既要求医学知识的严谨性，也需大众理解的易读性。任务分类覆盖五个大众核心健康领域，包括常见疾病科普，慢病管理科普，公共卫生科普，特殊人群科普，以及健康常识科普，确保测评全面覆盖大众健康需求场景。

MedBench评测榜单：该任务的评测采用评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：医学科普内容生成要求
answer：高质量医学科普内容
</p></div>
```

## 示例

```
<div><p><span>{</span>
  <span>"question"</span><span>:</span> <span>"目标受众：50-70 岁中老年高血压患者。

  科普主题：高血压日常管理，需包含饮食、运动、血压监测要点，澄清“血压正常就停药”的误区。

  内容要求：语言通俗易懂、方法可落地，字数 500 字内。"</span><span>,</span>
  <span>"answer"</span><span>:</span> <span>"高血压病友注意！3 个日常习惯帮你稳血压，别犯这些错

  不少老朋友查出高血压后，总问：“除了吃药，平时还能咋控血压？” 今天就说 3 个简单能做到的习惯，再提醒一个常见错。

  一、吃饭：少放盐，多吃 “降压菜”

  每天盐别超一个啤酒瓶盖(每日盐摄入量&lt;5克），像咸菜、酱肉这些腌制品要少吃。可以多吃芹菜、菠菜、冬瓜，比如早上喝碗芹菜小米粥，晚上炒盘菠菜，这些菜里的营养能帮着调血压，比吃保健品靠谱。

  二、运动：选 “慢动作”，别累着

  不用去健身房，每天早上或傍晚散步 30 分钟，走的时候别太快，能正常说话就行；也可以在家做太极、八段锦，这些动作慢，对膝盖好，还能让血管放松。记住：血压超过 160/100mmHg 时，先别运动，等降下来再说。

  三、测血压：固定时间，别乱比

  每天早上起床后 30 分钟（没吃药、没吃饭前）测一次，晚上睡前再测一次，用电子血压计就行。别跟邻居比血压值，每个人情况不一样，只要自己的血压稳定在 140/90mmHg 以下，就是好状态。

  重点提醒：别犯 “血压正常就停药” 的错！血压正常是药物和习惯共同作用的结果，随便停药会让血压忽高忽低，反而伤血管。有疑问一定要问医生，别自己调药。"</span>
<span>}</span>
</p></div>
```
