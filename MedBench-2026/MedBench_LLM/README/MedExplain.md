## 介绍

MedExplain数据集聚焦于评估模型将医疗术语解释给患者理解的医学释义的能力，需同时满足医疗术语的科学准确性与患者认知的通俗易读性，实现医生与患者沟通病情、诊疗方案、检查/治疗流程的需求，帮助患者清晰理解自身健康相关的专业概念，避免因术语壁垒导致的沟通误解。数据集任务覆盖患者诊疗全流程中的高频医疗术语场景，所有术语均源自真实门诊/住院病历、检查报告、诊疗方案中的核心表述，主要包括诊疗场景、用药场景、手术场景、康复护理场景等。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：标准术语
answer：术语解释
</p></div>
```

## 示例

```
<div><p><span>{</span>
  <span>"question"</span><span>:</span> <span>"待解释术语：冠状动脉造影
  
  目标受众：65 岁男性患者（疑似心绞痛，小学文化，担心检查痛苦/有风险）
  
  解释要求：说明检查目的、解释怎么做、疼不疼、有没有风险，语言口语化，避免未解释术语"</span><span>,</span>
  <span>"answer"</span><span>:</span> <span>"大爷，咱们说的‘冠状动脉造影’，其实就是给您心脏上的血管，这个咱们叫‘冠状动脉’，是给心脏供血的血管，拍个‘高清视频’，看看血管有没有堵、堵了多少，这样医生才能确定您胸痛的原因，好制定后续治疗方案。您躺在检查床上，医生会在您手腕或者大腿根的血管上扎个小针，跟平时输液差不多，会打麻药，扎针时就疼一下，然后通过这个小针送一根细细的管子到心脏血管附近，再打一点显影药能让血管在屏幕上显出来，同时机器会拍片子，整个过程大概30-60分钟，您全程是清醒的，不用开刀。扎针时打了麻药，基本不疼；推显影药时，可能会觉得胸口有点发热，这是正常的，不是疼。这个检查在医院做了很多年，很成熟，大多数人都没事。少数人可能对显影药有点反应（比如有点恶心），但很快就好，您不用太担心。检查前医生会再跟您详细说注意事项，有啥疑问您直接问就行。"</span>
<span>}</span>
</p></div>
```
