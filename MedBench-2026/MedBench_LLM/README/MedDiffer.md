## 介绍

MedDiffer数据集针对基层医疗场景下的基层全科医疗需求，聚焦于提升模型为基层医生提供差异化鉴别诊断建议的能力。覆盖呼吸系统常见病、消化系统多发病、社区获得性感染等基层常见病、慢性病，要求模型结合患者的信息和基础检查做出鉴别诊断以及鉴别诊断依据。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：患者信息和基础检查
answer：鉴别诊断以及鉴别诊断依据
</p></div>
```

## 示例

```
<div><p><span>{</span>
  <span>"question"</span><span>:</span> <span>"患者女性，45岁，因“上腹痛伴恶心2天”来诊。患者2天前进食油腻食物后出现上腹部持续性隐痛，呈阵发性加剧，偶向右肩背部放射。伴恶心、呕吐胃内容物一次。无发热、腹泻。既往体健，否认胃病史。查体：T 37.2℃，P 88次/分，BP 120/75mmHg。神清，皮肤巩膜无黄染。腹软，上腹部及右上腹有压痛，无反跳痛，墨菲氏征（+），肠鸣音正常。请给出需要鉴别的疾病及鉴别依据。"</span><span>,</span>
  <span>"answer"</span><span>:</span> <span>"鉴别诊断1：急性胃炎
  
  诊断依据1：多有饮食不当史，疼痛多位于上腹正中，压痛局限于上腹部，一般无放射痛，墨菲氏征阴性。
  
  鉴别诊断2：消化性溃疡
  
  诊断依据2：溃疡病多有慢性、周期性、节律性上腹痛病史。压痛点多位于上腹正中，位置固定，墨菲氏征阴性
  
  鉴别诊断3：急性胰腺炎
  
  诊断依据3：多由暴饮暴食或胆道疾病诱发，疼痛为持续性剧痛，常向腰背部呈束带状放射。查体压痛深在，可伴有反跳痛。需急查血清淀粉酶和脂肪酶明确。"</span>
<span>}</span>
</p></div>
```
