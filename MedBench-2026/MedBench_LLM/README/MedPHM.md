## 介绍

MedPHM数据集面向慢性病患者的长期健康管理需求，该数据集重点评估模型制定个性化健康管理方案的能力，覆盖高血压、糖尿病、慢性阻塞性肺疾病等常见慢性病的饮食运动方案、生活方式干预、急性加重预防、急性加重治疗、治疗效果管理等。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：患者当前健康状态
answer：健康管理方案
</p></div>
```

## 示例

```
<div><p><span>{</span>
  <span>"question"</span><span>:</span> <span>"患者男性，55岁，公司职员，新诊断“原发性高血压1级”，血压150/95mmHg。身高172cm，体重80kg。吸烟20年，10支/天，偶饮酒。久坐，无规律运动。请为其制定一份非药物健康管理方案，包含饮食、运动和治疗效果三个方面。"</span><span>,</span>
  <span>"answer"</span><span>:</span> <span>"饮食方案： 严格遵循低盐饮食（每日钠盐摄入&lt;5g），增加蔬菜水果、全谷物摄入，减少饱和脂肪和胆固醇摄入（DASH饮食原则）。
  
  运动方案： 每周进行至少5次、每次30-40分钟的中等强度有氧运动（如快走、慢跑、游泳），并可结合抗阻训练每周2次。
  
  生活方式干预： 戒烟，严格限制饮酒（男性每日酒精量&lt;25g），控制体重（目标BMI&lt;24kg/m²）。
  
  急性加重预防： 教育患者自我监测血压，识别并避免血压骤升的诱因（如情绪激动、突然用力、寒冷刺激）。
  
  治疗效果管理： 执行上述方案4周后复测血压，若血压仍未达标（&lt;140/90mmHg），则启动药物治疗。"</span>
<span>}</span>
</p></div>
```
