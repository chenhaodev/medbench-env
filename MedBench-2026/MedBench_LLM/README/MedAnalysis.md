## 介绍

MedAnalysis数据集为问答计算题，致力于患者的个性化风险预警与评估，重点评估模型对不同患者疾病进展风险、治疗并发症风险及预后不良风险的精准判断能力。覆盖常见的医学量表以及计算公式，综合评估患者的当前状态，给出对应结果。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：患者信息及评估量表
answer：量表结果
</p></div>
```
