## 介绍

MedCare数据集面向基层全科医生的日常诊疗工作，专注于评估模型提供适宜性诊疗处置建议的能力，涵盖常见病对症治疗、慢性病长期管理等。考察模型在门诊场景根据患者病历信息和基础检查信息，提供进一步诊疗方案，如药物治疗、手术治疗、进一步检验检查等结果的能力。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall结合评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：患者信息和基础检查
answer：诊疗处置建议
</p></div>
```

    <span>"answer"</span><span>:</span>   <span>"药物治疗：鼻内糖皮质激素：抗炎、抗水肿，按推荐剂量用，疗程≥12 周 。大环内酯类药物：14 元环大环内酯类（如克拉霉素小剂量，0.125g 每日 1 次 ），用于非变应性慢性鼻窦炎，抗炎兼免疫调节，疗程≥12 周 。抗菌药物：慢性鼻窦炎伴急性感染时，常规剂量用，疗程≤2 周 。黏液溶解促排剂：促进黏液排出、改善鼻腔鼻窦功能 。抗过敏药物：伴变应性鼻炎 / 哮喘者，可选口服（如开瑞坦 10mg 每日 1 次 ）或鼻用抗组胺药（如氮卓斯汀鼻喷剂 每鼻孔 2 喷 / 次 每日 2 次 ）、白三烯受体拮抗剂（如孟鲁司特 10mg 每日 1 次 ），疗程≥4 周 。减充血剂：鼻塞严重时，可选鼻用减充血剂短期用（疗程＜7 天 ） 。
