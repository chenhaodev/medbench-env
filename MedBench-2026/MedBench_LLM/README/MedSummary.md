## 介绍

MedSummary数据集聚焦于考察模型临床文档摘要生成能力，同时评估模型生成内容的临床医疗信息的完整性与摘要内容的简洁性，以满足医护人员快速获取患者核心医疗信息、辅助诊疗决策与病历管理的需求。数据集包含住院病历摘要、门诊病历摘要、检验检查报告摘要、手术记录摘要和出院小结摘要五大类。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：临床文档原始信息
answer：摘要内容
</p></div>
```
