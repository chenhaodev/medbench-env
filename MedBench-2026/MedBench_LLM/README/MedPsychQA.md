## 介绍

MedPsychQA数据集为适配大众对心理健康知识的科普需求，该数据集核心目标是评估模型对心理学知识的精准解答能力。该数据集形式为问答题，其知识覆盖基础心理学、发展心理学、临床心理学、心理生物学、人格心理学、认知心理学、应激与心身医学、心理评估、心理咨询与治疗、健康心理学等12个核心知识领域，细分场景更聚焦实用导向，包含常见心理障碍识别与成因、不同人群（儿童/中年/老年）心理发展特点、心理评估方法应用、心理咨询与治疗技术原理、应激管理与心身疾病干预、健康行为引导等多项细分问答场景。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：心理学知识问题
answer：答案
</p></div>
```
