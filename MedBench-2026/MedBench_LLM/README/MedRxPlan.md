## 介绍

MedRxPlan数据集面向患者的精准用药教育需求，该数据集形式为案例型问答题，内容覆盖各种常见疾病和相应的药物治疗方案，包含呼吸、循环、消化、泌尿、神经、内分泌与代谢、生殖、运动、免疫、血液等10大系统疾病，重点评估模型为不同患者提供个性化用药指导的能力，即根据患者基础信息、病史、检查结果及初始用药方案，评估模型分析用药合理性、判断疗效不佳或不良反应原因，并给出个性化用药调整方案的能力。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：患者基础信息、病史、检查结果及初始用药方案
answer：用药指导方案
</p></div>
```
