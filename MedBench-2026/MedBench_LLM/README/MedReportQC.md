## 介绍

MedReportQC数据集用于评估影像报告质控的CT报告质控任务，旨在评估模型在CT影像报告方面的质控能力，测试模型对常见疾病CT表现和诊断的了解，进一步缩减影像质控人员的时间。错误类型包含多字或少字、错别字、单位错误、方位错误、用词不当、检查项目与报告不符、描述与诊断不符、性别错误，为检测模型的全面能力，同时也包含无问题报告。其中错误类型的判断为限定域回答，对错误内容的修改建议为开放式回答。

MedBench评测榜单：该任务的评测采用Micro-F1结合基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

相关论文/技术报告：[Multimodal Human-AI Synergy for Medical Imaging Quality Control: A Hybrid Intelligence Framework with Adaptive Dataset Curation and Closed-Loop Evaluation](https://arxiv.org/abs/2503.07032)

## 元数据

数据集中包含以下信息

```
<div><p>question：影像报告内容
answer：错误类型及修改建议
</p></div>
```
