## 介绍

MedPrimary数据集面向基层全科医生的日常诊疗工作，专注于评估模型提供适宜性诊疗建议的能力，涵盖常见病对症治疗、慢性病长期管理等。考察模型在住院场景根据患者病历信息和基础检查信息，提供进一步诊疗方案，如药物治疗、手术治疗、进一步检验检查等结果的能力。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：患者信息和基础检查
answer：诊断（以及诊断依据）和治疗方案
</p></div>
```
