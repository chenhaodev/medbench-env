## 介绍

MedTeach数据集聚焦于评估模型基于患者真实病历生成临床教学案例的能力，需同时满足患者真实病历的信息完整性、隐私安全性与教学目标的针对性，实现医学生、规培医生通过真实临床场景培养临床思维、掌握诊疗规范、提升病例分析能力的教学需求。数据集任务覆盖临床核心诊疗场景，包含常见疾病诊疗教学案例、疑难病例分析教学案例、急救场景演练教学案例、医患沟通模拟教学案例和跨科室协作教学案例等5大类。

MedBench评测榜单：该任务的评测采用评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：门诊/住院病历、检查报告或手术记录等
answer：临床教学案例
</p></div>
```
