## 介绍

MedTreat数据集面向临床专病的精准诊疗方案制定，聚焦于评估模型在复杂临床场景下为特定疾病提供规范化、个性化诊疗建议的能力。数据集覆盖从常见慢性病到罕见病的多种疾病类型。模型需要依据患者的年龄、性别、疾病历史、症状、实验室检查等详细病历信息，进行深入、全面、系统的分析，给出初步诊断和包含一般治疗、药物治疗、手术治疗等相关的治疗方案。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。
