## 介绍

MedDiag数据集针对基层医疗场景下的全科诊断需求，聚焦于提升模型为基层医生提供精准诊断推荐的能力。覆盖呼吸系统常见病、消化系统多发病、社区获得性感染等基层常见病、慢性病，要求模型结合患者的信息和基础检查做出诊断以及诊断依据。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：患者信息和基础检查
answer：临床诊断以及诊断依据
</p></div>
```
