## 介绍

MedInsureCheck数据集旨在评估模型对医疗保险理赔单据的合规性与合理性进行自动化审核的能力。数据集基于仿真的医保结算数据构建，每条样本包含完整的诊疗信息链条，如费用明细。模型需根据给定的结构化信息，依据国家及地方《医保药品目录》、《医疗服务项目支付政策》等法规文件，对每笔费用进行多重审核判断。核心任务包括：药品与诊疗项目医保准入资格判定、限定支付范围合规性检查（如适应证、年龄、性别限制）、医疗行为合理性关联审核（如诊断与用药/检查是否匹配）以及重复收费、超标准收费等欺诈行为识别。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。
