## 介绍

DDx-advanced是专门为考验医学诊断能力而创建的数据集，难度高级。该数据集采用了多项选择题的题型设计。题干来自患者的真实病历信息，患者病历信息遵循2023年Surgical Case Report (SCARE)指南，主要包含：人口统计信息、症状表现、临床关注点、既往治疗和手术史、药物治疗史和过敏史、家族史、吸烟饮酒史、系统回顾。 模型需要对患者的病情特征描述进行深入、全面、系统的分析，并按照题目的要求，从选项中选择合适的答案（正确选项为一个或多个）。

MedBench评测榜单：该任务的评测采用精准度Accuracy作为评估指标。

MedBench评测榜单 2024年1月-2025年10月：该任务的评测采用Micro-F1作为评估指标。
