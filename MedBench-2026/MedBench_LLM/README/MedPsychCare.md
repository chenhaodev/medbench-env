## 介绍

MedPsychCare数据集为开放域问答题，聚焦于评估模型对常见心理问题的咨询解答与情绪疏导能力，覆盖常见心理问题咨询、特殊人群心理咨询、心理危机风险干预、跨场景心理需求、突发事件、慢性心理问题等不同类型，评估模型分析心理问题根源、判断情绪困扰核心诱因，并给出个性化咨询建议与情绪疏导方案的能力。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：心理健康问题咨询
answer：咨询解答与情绪疏导
</p></div>
```

## 示例

```
<div><p><span>{</span>
  <span>"question"</span><span>:</span> <span>"初二学生小妍，父母为让她 “不落后于同学”，除学校课程外，还为她报了数学、英语、物理 3 个补习班，周末及每晚课后都安排满学习任务。若小妍表现出疲惫不愿上课，母亲会说 “别人都在拼命补，你偷懒就会被淘汰”。近期小妍出现学习时注意力无法集中、经常莫名情绪低落、甚至故意找借口逃避去补习班的情况。结合该案例，说明临床心理学中可采用哪些干预方法帮助小妍？"</span><span>,</span>
  <span>"answer"</span><span>:</span> <span>"针对小妍的情况，临床心理学可采用以下干预方法：一是认知行为疗法（CBT），先帮助小妍识别不参加补习班 = 会被淘汰的不合理认知，通过梳理未额外补习但成绩稳定的同学案例，以及过度补习导致效率下降的经历等证据，打破负面思维定式，再结合时间管理训练，协助小妍合理规划学习与休息时间，改善注意力不集中的问题；二是家庭治疗，邀请小妍父母参与咨询，通过沟通让他们意识到过度补习 + 否定式激励给小妍带来的心理负担，引导父母调整教育方式，从强制报班转变为尊重小妍的学习节奏，根据她的精力与需求精简补习课程，营造宽松的家庭学习氛围；三是情绪焦点疗法（EFT），引导小妍表达内心的疲惫、低落等情绪，帮助她理解情绪背后的压力来源（如学业任务过载、父母的过高期待），通过情绪疏导技巧（如情绪日记、积极自我对话），缓解负面情绪，减少因情绪困扰产生的补习班逃避行为。"</span>
<span>}</span>
</p></div>
```
