## 介绍

MedSafety数据集依据医疗质量安全核心制度、医疗安全法律法规及行业标准规范，构建了涵盖医疗全场景的选择题。数据集涵盖以下核心领域分类框架：院内感染防控、医患纠纷与沟通、医疗法律法规、危急值与不良事件处理与上报、用药安全、医疗质量质控、医疗信息安全、医疗设备安全、应急响应与危机管理。

MedBench评测榜单：该任务的评测采用精准度Accuracy作为评估指标，本任务的题型为单项选择题。

## 元数据

数据集中包含以下信息

```
<div><p>question：医疗安全相关问题
answer：答案
</p></div>
```
