## 介绍

MedTerm数据集面向医学专业人士，要求基于提问问题，对临床 / 科研核心医学名词进行精准释义。释义需严守权威医学标准，清晰阐述名词定义、生理/病理机制、心电图等关键特征及临床诊断、治疗决策中的应用价值，兼顾理论深度与实践关联性，语言精简且逻辑连贯，为专业诊疗、教研及学术交流提供有效支撑。任务涵盖基础医学、临床诊断、疾病病理、治疗干预、预后与循证医学等5大类高频术语。

MedBench评测榜单：该任务的评测采用基于要点信息计算Macro-Recall以及评价模型LLM-as-a-Judge作为评估指标。
