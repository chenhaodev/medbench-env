## 介绍

MedHG是专门用于评测模型导诊能力的数据集，为医患之间关于症状、病史、医疗诉求等内容的对话。模型需要分析对话信息，根据患者的症状和体征，为其推荐**给定范围内**合适的挂号科室。由于对话信息同时存在多个疑似诊断，因此输出需要覆盖所有诊断的对应科室。科室范围包含35个常见科室以及16个儿科专科科室，比如普外科、呼吸内科、心血管内科、消化内科、小儿血液肿瘤科、小儿感染科等。MedHG中症状相关的疾病谱覆盖了各年龄段，可以有效地评估模型在实际导诊应用中的性能。

MedBench评测榜单：该任务的评测采用精准度Accuracy作为评估指标。

MedBench评测榜单 2024年6月-2025年10月：该任务的评测采用Micro-F1作为评估指标。

MedBench评测榜单 2024年1月-5月：只有当模型输出的答案与正确答案完全一致时，该题才算作答正确。该任务的评测采用Accuracy作为评估指标。

## 元数据

数据集中包含以下信息

```
<div><p>question：患者信息
answer：输出全部可能的科室
</p></div>
```
